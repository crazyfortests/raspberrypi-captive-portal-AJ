# 📘 Project Setup & Usage Guide

## 🛠️ UI Modifications
- For UI changes, edit:
  - `/public/index.html`
- For backend response handling:
  - `/src/`

---

## ⚙️ Configuration (Important)

Update the following values in the script as per your requirements:

```bash
export WifiSSID=_MEIC_
export WifiPasswd=Attherate123_
export Host_name=microsoft.com
export MY_VAR=my_value
```

## Running script

Simply run 

```bash
sh run.py
```
